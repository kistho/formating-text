Apa itu HTML

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag Penanda atau markup ini, nanti akan kita sebut dengan HTML berperan untukmenentukan struktur konten dantampilan dari sebuah web Kalau kita ibaratkan nih..HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar. HTML memiliki kelebihan seperti memiliki sumber komunitas yang cukup besar dengn pengguna yang luas Tidak hanya itu, kelebihan lainnya adalah HTML mudah untuk dipelajari oleh seorang Web devoleper pemula Kelebihan lainnya adalah HTML merupakan bahasa pemrograman yang gratis atau open source
